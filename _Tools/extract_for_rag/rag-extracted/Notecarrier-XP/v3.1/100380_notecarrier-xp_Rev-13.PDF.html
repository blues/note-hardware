Named Nets / Signals

Net/Signal Connections / Notes
+VUSB Shown connected to USB C, ORING, and PIN HEADER.
+VBAT Shown connected to ORING and PIN HEADER.
+VMAIN Shown as ORING output; connected to Buck/Boost 3V8/3V3, Buck/Boost 3V3, and PIN HEADER.
+VMODEM Shown as Buck/Boost 3V8/3V3 output to M.2 CONNECTOR KEY E.
+VIO Shown as Buck/Boost 3V3 output to M.2 CONNECTOR KEY E and PIN HEADER.
VIO_EN Shown between M.2 CONNECTOR KEY E and the buck/boost regulator control path.
VMODEM_SEL Shown between M.2 CONNECTOR KEY E and the buck/boost regulator control path.
+VACT_GPS_IN Shown from PIN HEADER toward M.2 CONNECTOR KEY E.
+VACT_GPS_OUT Shown from M.2 CONNECTOR KEY E toward PIN HEADER.
I2C Shown in interface bus between M.2 CONNECTOR KEY E and PIN HEADER.
