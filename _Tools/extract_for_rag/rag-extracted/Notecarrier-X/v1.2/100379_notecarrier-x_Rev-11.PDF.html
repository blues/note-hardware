Sheet Section: Battery Charger w/ Power Path

Battery Charger w/ Power Path Design Notes

  • Charger: Default charging voltage: 4.2V
  • Charger: Default charging current: 320mA
  • Charger: Default termination current: 20mA
  • Charger: Default pre-charge current: 20mA
  • Charger: No battery thermal protection
  • Power Path: Input current limit: 3.5A
  • Power Path: 26mOhm reverse blocking transistor
  • Power Path: 15mOhm battery transistor
  • Buck Regulator: 1.5MHz frequency
  • Buck Regulator: Input Voltage: 5V
  • Buck Regulator: Output Voltage: 3.3-4.2V
  • Buck Regulator: Peak current: 2.5A
